## Set up the project

- Install dependencies:
```sh
$ yarn
```

- Start the localhost:
```sh
$ yarn run dev
```

- Set up the initial values in ``src/helpers/room/room.types.ts``
- Change language by pressing the button below the heading.

- Run tests:
```sh
$ yarn run test
```
